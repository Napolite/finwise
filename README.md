Budgeting App

A simple, no-nonsense budgeting app designed to help you understand where your money goes—and take control of it.

This app isn’t about shaming spending or forcing extreme rules. It’s about clarity. When you can see your money clearly, better decisions follow. Always has been that way.

📌 Features

Income Tracking
Log multiple income sources with dates and categories.

Expense Management
Track daily, weekly, or monthly expenses with clear labels.

Category Budgets
Set limits per category (food, rent, transport, etc.) and stay accountable.

Transaction History
A clean, chronological view of all financial activity.

Spending Insights
See where your money leaks and where it behaves.

Local-First (Optional)
Your data stays yours unless you choose otherwise.

🧭 Philosophy

Budgeting works best when it’s boring.

No gimmicks.
No fake “financial freedom” slogans.
Just accurate numbers and honest feedback.

This app follows a simple rule: awareness precedes discipline.

🛠 Tech Stack

Frontend: React / React Native

Backend: Node.js

State Management: Local state + persisted storage

Styling: Tailwind / NativeWind

Data Storage: Local storage / Database (optional)
